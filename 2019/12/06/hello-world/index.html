【题外话】

迟早有一天我会弄混各种语言的语法

以获取字符串的某个字符（下称为“某一项”）为例

C

先构造字符串再输出某一项

C语言的字符串用方括号来得到某一项

————————————

也可以直接对引号括起来的字符串常量用方括号来得到某一项

==============================

java

先构造字符串再输出某一项

java的字符串不能用方括号，要用.charAt()来得到某一项

————————————

也可以直接对引号括起来的字符串常量用.charAt()

是charat()不是getchar()！！！

==============================

py

先构造字符串再输出某一项

与C语言类似双引号括起来的字符串可以用方括号来得到某一项

————————————

也可以直接对引号括起来的字符串常量用方括号来得到某一项

==============================

IDL

惊人地与java一致，是使用.charAt()来得到某一项

————————————

惊人地与java不一致（……）

不可以直接对引号括起来的字符串常量用.charAt()
